# Customer Retention & Churn Analysis – Task 2

## 📊 Project Overview
This project analyzes customer churn patterns in a subscription-based telecom business using Power BI.

The objective is to identify churn drivers, retention opportunities, and customer lifetime trends.
 
 ## 📈 Key KPIs

- Total Customers: 7,032
- Churn Percentage: 26.6%
- Revenue Lost Due to Churn: 2.86M
- Average Tenure: 32.42 months

## Key Insights

- Month-to-month contracts show the highest churn.
- Customers using Electronic Check have higher churn probability.
- Fiber optic internet users show elevated churn risk.
- Longer contract durations significantly reduce churn.

## 💡 Business Recommendations

1. Encourage customers to switch to long-term contracts.
2. Provide discounts for automatic payment methods.
3. Introduce loyalty benefits for high-risk customers.
4. Improve service quality for fiber optic users.

## 🛠 Tools Used

- Power BI
- DAX Measures
- Data Cleaning
- KPI Analysis
